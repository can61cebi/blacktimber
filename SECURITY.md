# Security Policy

## Supported versions

The latest release is the supported version. Older versions are not patched.

## Reporting a vulnerability

If you find a security issue, please do not open a public issue. Instead, report it
privately through the repository's
[security advisories](https://github.com/can61cebi/blacktimber/security/advisories/new),
or contact the maintainer directly. You can expect an initial response within a few days.

BlackTimber has no network surface of its own: it adds no listeners, no web server and no
external calls. The most relevant concerns are server side griefing and protection bypass,
so reports about felling protected blocks or escaping the build checks are especially
welcome.
